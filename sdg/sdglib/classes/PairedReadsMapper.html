PairedReadsMapper¶

class PairedReadsMapper¶

A mapper for linked reads from a PairedReadsDatastore. Supports partial remapping of unmapped reads or of a selection list.

void map_reads(std::unordered_set<uint64_t> const &reads_to_remap = {})¶

Maps each read in the data-store to the nodes using unique kmers form the graph.

Maps each read in the data-store to the nodes using unique kmers form the graph. Reads are mapped only if the placing in the graph is unique, if one or more kmers of a read maps to a different node the mapping for that read is discarded. Results are stored in this.read_to_node and this.reads_in_node

If a set of reads ids is passed the mapper will only remap those reads.
