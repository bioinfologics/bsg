DistanceGraph¶

class DistanceGraph¶

This is a description of the graph creating new linkage, the nodes are referenced from the original graph.

The LinkageDIgraph has links describing structure in the graph but not the nodes, the nodes are references from sg original graph, the linkage digraph references to the nodes of a sequence digraph and only contains the links. It’s usually used to encode solutions from algorithms.

Each link is represented on each node it appears as -A,B on A and -B,A on B

Subclassed by SequenceDistanceGraph
